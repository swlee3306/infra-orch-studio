# Documentation Map

이 문서는 `docs/` 안의 문서를 두 종류로 구분한다.

- `active source of truth`
  - 현재 구현, 운영, 계약을 설명하는 문서
- `historical snapshot`
  - 특정 시점의 진단, 감사, 개선 계획을 기록한 문서
  - 현재 구현과 다를 수 있으며, 최신 계약 문서로 읽으면 안 된다
